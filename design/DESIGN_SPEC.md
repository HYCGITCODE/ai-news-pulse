## 1. 设计概览

### 1.1 设计理念
- **功能至上**: 每个视觉元素都服务于核心功能
- **清晰明确**: 信息层级分明，用户一眼找到重点
- **极致简洁**: 去除所有不必要的装饰

### 1.2 参考风格
- Linear - 极简专业感
- Stripe - 清晰的信息架构
- Apple - 精致的细节处理

## 3. 组件详解

### 3.1 新闻卡片 (News Card)

**尺寸**: 
- Desktop: 宽度 100%, 高度自适应 (min 120px)
- Mobile: 宽度 100% - 32px padding

**结构**:
```
┌─────────────────────────────────────────┐
│  [Tag Icon]  Title (18px, Bold)         │
│                                          │
│  Summary text (14px, secondary color)   │
│  Limited to 2 lines with ellipsis       │
│                                          │
│  Source • Relative Time (12px, tertiary)│
└─────────────────────────────────────────┘
```

**状态说明**:

| 状态 | 背景色 | 边框 | 阴影 | 其他 |
|------|--------|------|------|------|
| **Default** | #FFFFFF | 1px solid #E5E7EB | 0 1px 2px rgba(0,0,0,0.05) | - |
| **Hover** | #F9FAFB | 1px solid #D1D5DB | 0 4px 6px rgba(0,0,0,0.1) | translateY(-2px) |
| **Active** | #F3F4F6 | 1px solid #3B82F6 | 0 1px 2px rgba(0,0,0,0.05) | - |
| **Loading** | #F9FAFB | 1px solid #E5E7EB | none | opacity: 0.6 |

**深色模式**:
- Default: bg #1F2937, border #374151
- Hover: bg #374151, border #4B5563

---

### 3.2 来源标签 (Source Tag)

**尺寸**: 高度 32px, padding 0 12px

**结构**:
```
┌─────────────────┐
│  TechCrunch AI  │
└─────────────────┘
```

**状态说明**:

| 状态 | 背景色 | 文字色 | 边框 |
|------|--------|--------|------|
| **Default** | #F9FAFB | #6B7280 | 1px solid #E5E7EB |
| **Hover** | #3B82F6 | #FFFFFF | 1px solid #3B82F6 |
| **Active** | #3B82F6 | #FFFFFF | 1px solid #3B82F6 |
| **Disabled** | #F3F4F6 | #9CA3AF | 1px solid #E5E7EB (opacity: 0.5) |

**交互**:
- Click: 过滤新闻列表
- Active 状态表示当前选中的筛选条件
- "全部" 标签始终显示在最左侧

### 3.3 头部导航 (Header)

**尺寸**:

**结构**:
```

┌─────────────────────────────────────────┐
│  ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓       │  (Title - 60% width)
```

- 颜色：#E5E7EB → #F3F4F6 → #E5E7EB

## 4. 深色模式完整设计

### 4.1 颜色映射

| 元素 | 浅色模式 | 深色模式 |
|------|----------|----------|
| 背景主色 | #FFFFFF | #111827 |
| 背景次级 | #F9FAFB | #1F2937 |
| 文字主色 | #111827 | #F9FAFB |
| 文字次级 | #6B7280 | #9CA3AF |
| 边框 | #E5E7EB | #374151 |
| 强调色 | #3B82F6 | #60A5FA |

### 4.2 特殊处理

- 阴影在深色模式下使用更低透明度
- 卡片悬停效果使用背景色变化而非阴影
- 确保对比度符合 WCAG AA 标准 (4.5:1)

## 5. 响应式断点

### 5.1 断点定义

| 断点 | 宽度范围 | 布局变化 |
|------|----------|----------|
| Mobile | < 768px | 单列布局，筛选栏可横向滚动 |
| Tablet | 768px - 1023px | 单列布局，筛选栏完整显示 |
| Desktop | ≥ 1024px | 可考虑双列布局 (可选) |

### 5.2 移动端适配要点

1. **字体大小**: 最小 14px (正文), 16px (标题)
2. **点击区域**: 最小 44px × 44px
3. **横向滚动**: 禁止页面横向滚动
4. **筛选栏**: 支持横向滚动，显示更多来源标签
5. **卡片间距**: 适当减小 (16px → 12px)

| 焦点状态 | 所有交互元素有明确 focus 状态 | 手动测试 |

## 8. 特殊交互说明

### 8.1 来源筛选交互
- 点击来源标签 → 立即过滤列表 (无页面刷新)
- 再次点击同一标签 → 取消筛选 (显示全部)
- 点击"全部" → 清除所有筛选

### 8.2 主题切换
- 点击太阳/月亮图标 → 立即切换主题
- 记住用户偏好 (localStorage)
- 切换动画：< 100ms, 无闪烁

### 8.3 无限滚动
- 滚动到底部前 200px 开始加载
- 显示加载骨架屏
- 加载失败显示重试按钮
